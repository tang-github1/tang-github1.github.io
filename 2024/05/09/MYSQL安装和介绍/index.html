MYSQL安装

安装包下载：https://mysql.com/downloads 下载mysql-5.7.x.tar

启动服务并开机自启动

[root@db1 ~]# systemctl start mysqld
[root@db1 ~]# systemctl enable mysqld

查看进程和端口

[root@db1 ~]# ps -C mysqld
[root@db1 ~]# ss -lanpt | grep mysqld
LISTEN 0 80 [::]:3306 [::]:* users:((“mysqld”,pid=84196,fd=22))

查看初始密码

[root@db1 ~]# grep password /var/log/mysqld.log
2024-02-01T08:20:42.033188Z 1 [Note] A temporary password is generated for root@ localhost: fM&2B8>FHuZ:

初始密码无法登录解决：停止并删除初始目录，重启会重新创建文件和密码

/etc/init.d/myqld stop
rm -rf /var/lib/mysql/*
systemctl start mysqld

方式2：使用mysqladmin命令进行修改

mysqladmin -uroot -p’初始密码/旧密码’ password ‘新密码’

[root@db1 ~]# mysqladmin -uroot -p’fM&2B8>FHuZ:’ password ‘Mysql12#$’
